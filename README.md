# CIAO Github Pages Website

This website is a proof of concept for the CyberInfrastructure Alliance of Oregon organization whose goal is to unite Oregon research 
schools to facilitate collaboration and share resources. 

## What is this? 

This webapp is built using React Router v7 to create a aesthetically pleasing webpage that does not need to be hosted from a 
Node.js backend server and can be hosted directy from the github repo via Github Pages service. Do to this, the webap is entirely 
static and information cannot be displayed dynamically (no live stats such as cluster utilization, available services, user count, etc) 
but the files are structured in a way that can be easily followed and edited by someone with minimal typescript skills.
